## 画面遷移
 
### 顧客向け画面遷移
- トップページ -> 商品一覧 
- トップページ -> 会員登録 -> メール認証画面
- トップページ -> ログイン 
- トップページ -> 特定商取引法表示 
- トップページ -> プライバシーポリシー 
- トップページ -> FAQ 
 
- 商品一覧 -> 商品詳細 
- 商品詳細 -> カート 
 
- カート -> 注文情報入力 
- カート -> ログイン（任意） 
 
- 注文情報入力 -> 注文確認 
- 注文確認 -> 注文完了 
 
- 注文完了 -> トップページ 
- 注文完了 -> マイページ（ログイン中の場合） 
 
- 会員登録 -> トップページ（ログイン状態で遷移） 
- ログイン -> トップページ（または元のページへ戻る） 
- ログアウト -> トップページ 
 
- マイページ -> 注文履歴 
 
### 管理者向け画面遷移
- 管理者ログイン -> 商品管理（一覧） 
- 管理者ログイン -> 注文管理（一覧） 
- 管理者ログイン -> 顧客管理
 
- 商品管理（一覧） -> 商品登録 
- 商品管理（一覧） -> 商品編集 
 
- 注文管理（一覧） -> 注文詳細 
 
- 管理者ログアウト -> 管理者ログイン
